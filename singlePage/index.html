나루터?

호스팅 세팅이 필요없는 갠홈 솔루션 나루(naru.pub)에 바로 업로드하여 사용할 수 있는 비설치형 홈페이지 템플릿입니다. 현재 나루에서는 DB 기능을 사용할 수 없어 게시글 관리 기능은 개발되지 않았습니다.

현재 보고 계신 페이지는 나루터의 첫 페이지가 되는 곳입니다. 이곳에 페이지 소개를 작성하거나 이미지를 추가해보세요.

원하는 만큼 html 파일에서 모듈 박스를 추가하고 내용을 자유롭게 작성하세요.

활용할 수 있는 모듈의 종류는 Board 페이지를 확인하세요.

본 템플릿은 크로미움 기반 브라우저를 기준으로 제작 되었습니다.

텍스트 모듈

div.text-module.module

텍스트를 입력할 때 사용하는 모듈입니다.

모듈을 조합하고, 제목용인 h1 ~ h6 태그와 p 태그를 활용해 글을 작성해보세요.

h1 태그

h2 태그

h3 태그

h4 태그

h5 태그
h6 태그

다이어리 모듈

div.diary-module.module

이미지 1장과 텍스트를 입력할 때 사용하는 모듈입니다.

날짜 입력란을 갖고 있으며 날짜는 모듈 오른쪽 상단에 나타납니다.

24.12.25

임베드 모듈

div.embed-module.module / div.embed-module.full.module

embed 태그를 통해 외부 사이트 콘텐츠를 가져올 때 사용하는 모듈입니다.

다른 모듈과 동일한 여백값을 갖지 않고 Embed 페이지처럼 화면을 꽉차게 사용하려면 full 클래스를 추가합니다.

샘플은 각각 오이카페(oeee.cafe)와 타이포블루(typo.blue)의 대표 콘텐츠입니다.

배너를 위한 모듈은 총 3가지로 준비하였습니다.

모든 배너 이미지는 직접 업로드하셔야하고 배너주소 또한 직접 작성해주셔야합니다.

개인 배너 모듈

div.banner-module.self.module

자신의 배너 정보를 등록하는 영역입니다. 주소 복사 기능이 제공됩니다.

배너 그룹 모듈

div.banner-module.group.module

배너들을 일정한 그룹으로 묶을 때 사용하는 모듈입니다. 제목이 제공됩니다.

일반 배너 모듈

div.banner-module.module

기본적으로 배너들을 나열해 두는 영역입니다.

갠홈용 배너 사이즈는 가로 200 x 세로40 px .. 가 전통적으로 이어지고 있는 듯합니다.
